# algorithm-analyzer
Algorithm visualizer illustrates how algorithms work in a graphical way. It mainly aims to simplify and deepen the understanding of algorithms operation. Within the paper we discuss the possibility of enriching the standard methods of teaching algorithms, with the algorithm visualizations

Algorithm-visualizer is a program written in Python. Program executes and it provides graphical window. Algorithms contains visualizations of algorithms, each element is represented as a bar graph which is randomly generated and when program is executed it sorts each element(graph) in either ascending way or descending way followed by algorithm technique chosen by user.
